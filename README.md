# Search_engine

## Описание приложения
*search server* - это консольное приложение, написанное для быстрого поиска набора слов среди документов. Получает на вход данные в виде json-файлов. 

## Сборка
Приложение можно собрать через cmake + make. После загрузки файлов:

`cd /path/to/directory`

`mkdir ./build`

`cd ./build`

`cmake ../`

`make`

После этого приложение собрано, может быть запущено:

`./search_server`

## Использование
Для настройки параметров приложения нужно редактировать файлы:
`/src/config.json` - изменение документов, по которым будет идти поиск и максимального количества ответов.

`/src/requests.json` - изменение слов, которые необходимо найти.
