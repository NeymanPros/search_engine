# Search_engine

## Описание приложения
*Search_engine* - это консольное приложение, написанное для быстрого поиска набора слов среди содержимого текстовых файлов. Получает на вход данные в виде json-файлов. 

## Сборка
Приложение можно собрать через cmake + make. После загрузки файлов:

`cd /path/to/directory`

`mkdir ./build`

`cd ./build`

`cmake ../`

`make`

После этого приложение собрано, может быть запущено:

`./search_engine`

## Использование
Для настройки параметров приложения нужно редактировать файлы:

`resources/config.json` - здесь отображаются файлы, по которым будет вестись поиск, а также максимальное количество ответов.

`resources/requests.json` - указывается запрос, а именно какие слова необходимо найти.
